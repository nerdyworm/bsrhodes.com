Pooky CMS

A while back I decided to convert my wordpress blog to my own personal rails project. Well this project has evolved into what I now call pooky.

Pooky is powered by Rails, MongoDB, GridFS, and Liquid Templates. This allows me the flexibility to customize every editor field to suit the needs of the content. By leveraging the power of a document oriented database I am able to store things in a logic manor without the burden of a ridiculous relational database join system.

Smrks

smrks is a simple to use and organize bookmark site. This project was aimed at creating a private bookmarking site that didn’t have frivolously features of social networks. The best feature of the project is the bookmarklet. It allows you to bookmark a site without having to open a new window.

Pooky Ping

pooky ping is a basic http request based monitor system. it tracks parameters like response time that follows the health of your server. it organizes all the data it collects into easy-to-read graphs, and even notifies the administrator by email if the server becomes unresponsive. we use simple monitor on our website and any we administrate.
